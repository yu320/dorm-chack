# 圖片文字辨識與自動改名工具 (專業視窗版)

這是一個具備 **Windows 11 現代化深色介面 (Fluent Design)** 的本地端自動化工具。它可以幫您平行掃描大量圖片中的文字，並自動將圖片「以其內含的文字重新命名」，徹底省去手動整理照片的麻煩。

Copyright © @youzih

## ✨ 核心特色
- 🖥️ **原生視窗介面**：不再受限於網頁瀏覽器，可自由點擊選擇電腦中任意的來源資料夾與輸出資料夾。
- 🚀 **多執行緒平行加速**：同時開工處理多張圖片，把您的 CPU 榨乾，大幅縮短處理時間。
- 🤖 **智慧雜訊過濾**：自動過濾亂碼與無意義符號，確保重新命名的檔名乾淨、合法。
- 🧠 **自動記憶 (Hash Cache)**：自動比對 MD5，重複的圖片瞬間跳過，處理上千張圖也不怕中斷重來。
- 📝 **完整辨識日誌**：系統自動產出附帶時間戳記的 `.txt` 紀錄檔，方便您複製裡面的文字或後續建檔。

---

## 🚀 如何使用 (免安裝，點擊即用)

為了讓任何人都能無痛使用，本工具內建了「全自動環境建置」功能。您**完全不需要安裝 Python**，只要連上網路並點擊以下檔案即可：

### 1. 啟動方式選擇
資料夾內提供了三種啟動方式，請根據您的需求選擇**其中一種**點擊兩下：

- 👉 **`啟動介面.vbs` (強烈推薦)**：
  - 特色：**完全無痕隱藏黑視窗**，只會優雅地彈出美麗的主介面。
  - *(注意：若是全新電腦「第一次」執行本軟體，請先使用下方的 `.bat` 啟動一次讓它下載環境，之後就可以永遠使用 `.vbs` 啟動了！)*

- 👉 **`Run_CPU.bat` (一般電腦版)**：
  - 特色：相容性 100%，任何 Windows 電腦（包含一般筆記型電腦）皆可運行。第一次啟動會顯示黑色視窗並自動下載約 150MB 的核心引擎，啟動完成後黑視窗會自動關閉。

- 👉 **`Run_GPU.bat` (NVIDIA 顯卡加速版)**：
  - 特色：專門為擁有 NVIDIA 獨立顯示卡的玩家設計。第一次啟動會自動下載高達 2.5GB 的 CUDA 運算引擎（請耐心等候）。一旦安裝完成，辨識速度將獲得飛躍性的提升。

### 2. 開始處理
1. 在彈出的漂亮視窗中，點擊 **「瀏覽...」** 選擇您要處理的圖片資料夾。
2. 選擇你要將改名後的圖片輸出到哪裡。
3. 選擇 **「複製」** (保留原檔) 或 **「移動」** (清空原圖)。
4. 點擊 **🚀 開始執行辨識與改名**，接著您就可以在下方的黑色方塊中看著即時進度條狂奔了！

## 📂 處理結果說明

前往您剛剛指定的「輸出資料夾」，您會看到：
- **📁 已辨識/**：成功抓出文字的圖片都會在這裡，且已經被重新命名好了。
- **📁 未辨識/**：如果圖片太模糊或沒有文字，會被歸類到這裡（檔名開頭會加上 `未辨識_`）。
- **📄 [時間]-辨識紀錄.txt**：所有被成功辨識出的文字紀錄（包含原檔名與完整內容）。

## 💻 技術架構 (開發者參考)
- **環境管理**：透過 Astral `uv` 進行極速虛擬環境自動構建。
- **UI 框架**：Python 原生 `tkinter` 搭配開源 `sv-ttk` (Sun Valley Theme) 主題渲染。
- **AI 核心**：PyTorch + EasyOCR。
- **相依套件**：定義於 `.bat` 的啟動腳本與虛擬環境中，達成開箱即用的隔離環境體驗。

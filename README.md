# 圖片文字辨識與自動改名工具

這是一個可以幫您掃描圖片中的文字，並自動將該圖片「以圖片內的文字重新命名」的自動化工具。

## 🚀 如何使用 (給一般使用者的指南)

1. 將你想改名的圖片（支援 `.jpg`, `.png`, `.jpeg`），放入 `source_images` 資料夾中。
2. 點擊兩下執行 `run.bat`。
3. 程式會自動開始辨識，請稍微等它一下。
4. 處理完成後，請到 `processed_images` 資料夾內收取結果：
   - **已辨識**：成功辨識的圖片會放在這裡，並已重新命名。
   - **未辨識**：找不到文字的圖片會放在這裡，保留原檔名並加上「未辨識_」前綴。
   - 📝 **[時間]-辨識紀錄.txt**：系統會在此資料夾自動產生一個文字檔（檔名包含當次執行的時間），詳細記錄「每張原始圖片」與對應的「完整辨識文字」，方便您後續複製或建檔！

### ⚠️ 注意事項
- **首次執行**：因為電腦需要自動下載 Python 與 AI 辨識模型，可能會花費 3 到 5 分鐘的時間（取決於您的網路速度），請耐心等候，直到終端機畫面出現「處理完畢」即可。
- **保留原檔**：本工具預設為「複製」模式，您放在 `source_images` 的原圖不會消失，以確保資料安全。
  - 如果想要改成「移動」模式（處理完自動清空原圖），請用文字編輯器打開 `src/config.py`，將 `MOVE_FILES = False` 改成 `True` 即可。

## 💻 針對開發者的指南 (Development)
本專案使用 `uv` 進行依賴管理與執行。

```bash
# 執行主程式
uv run main.py
```
> 依賴套件 (EasyOCR, PyTorch) 已透過 PEP 723 inline script metadata 定義在 `main.py` 頂部，不需額外初始化環境。

### 🏎️ 開啟 GPU 硬體加速 (選用)
為了確保程式在所有電腦上都能無痛執行，預設安裝的是「CPU 版本」的引擎。
如果您擁有 **NVIDIA 顯示卡 (CUDA)**，並希望大幅提升辨識速度，請在終端機執行以下指令來安裝 GPU 專用版本：

```bash
uv add torch torchvision --index-url https://download.pytorch.org/whl/cu121
```
*(注意：此安裝檔約 2.5GB，下載時間較長。安裝完成後，程式便會自動調用顯卡運算！)*
